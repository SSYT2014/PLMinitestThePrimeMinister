実行方法（実行ファイルの動かし方）

実行するためのコマンドは以下の通りである。

java -jar primeministers.jar

Mac OS Xの場合には、アプリケーションPrimeMinistersByJava.appをダブルクリックするほうが簡便である。

PrimeMinisters.app

また、openコマンドでPrimeMinistersByJava.appを開いても実行可能である。

open PrimeMinistersByJava.app
